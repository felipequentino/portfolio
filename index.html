Destaque: Óusi Featured: Óusi

Sistema Avançado de Monitoramento

Advanced Monitoring System

Este projeto representa um avanço significativo na tecnologia de segurança, utilizando técnicas de deep learning de última geração para detectar armas em transmissões de vídeo em tempo real com precisão excepcional.

This project represents a significant advancement in security technology, utilizing state-of-the-art deep learning techniques to detect weapons in real-time video streams with exceptional accuracy.

O sistema foi treinado em um conjunto diversificado de dados de armas em várias condições de iluminação, ângulos e obstruções parciais. Ao implementar transferência de aprendizado e técnicas personalizadas de aumento de dados, o modelo atinge 95% de precisão com uma taxa de falsos positivos notavelmente baixa.

The system was trained on a diverse dataset of weapons in various lighting conditions, angles, and partial occlusions. By implementing transfer learning and custom data augmentation techniques, the model achieves 95% accuracy with a remarkably low false positive rate.

Além disso, o Óusi é integrado ao Reconhecimento Facial Aprimorado com rastreamento de pessoas, formando uma solução completa de segurança.

Also, Óusi is combined with the Enhanced Facial Recognition and person tracking, making it a complete security software.

Principais conquistas técnicas incluem:

Key technical achievements include:

  • Arquitetura YOLOv8 personalizada com modificações para melhorar a detecção de objetos pequenos Custom YOLOv8 architecture with modifications for improved small object detection
  • Pipeline de inferência otimizado para processamento em tempo real a mais de 10 FPS em hardware do cliente Optimized inference pipeline for real-time processing at 10+ FPS on consumer hardware
  • Implementação de verificação de consistência temporal para reduzir falsos positivos Implementation of temporal consistency checking to reduce false positives
  • Capacidades de implantação em dispositivos de borda para sistemas com conectividade limitada Edge deployment capabilities for systems with limited connectivity
